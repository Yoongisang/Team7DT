## 3. CSV 컬럼 정의 (13개)

| 컬럼 | 단위 | 설명 |
|------|------|------|
| `Timestamp` | s | 시뮬레이션 경과 시간 |
| `World_X`, `World_Y`, `World_Z` | cm | UE 월드 좌표 |
| `UTM_Easting`, `UTM_Northing` | m | UTM 변환 좌표 |
| `UTM_Zone` | int | UTM 구역 (한국 = 52) |
| `Velocity_kmh` | km/h | 차량 속도 |
| `Yaw` | ° | 차량 헤딩 (월드 좌표계) |
| `Throttle` | 0~1 | SplineFollower 가속 입력 |
| `Steering` | -1~1 | SplineFollower 조향 입력 |
| `Brake` | 0~1 | SplineFollower 제동 입력 |
| `Acceleration_kmh_per_s` | km/h/s | 직전 프레임 대비 속도 변화 |

## 4. 데이터 샘플

### 시작 부분 (출발 직후)
```csv
Timestamp,World_X,World_Y,World_Z,UTM_E,UTM_N,Zone,Speed,Yaw,Throttle,Steering,Brake,Accel
0.408,-3490.00,-9330.00,54.51,320797.7602,4039142.5139,52,1.47,0.0000,1.000,0.092,0.000,0.00
```
→ **Throttle 100% 풀 가속**, 속도 0에서 출발

### 중간 부분 (코너 진입)
```csv
77.512,10256.78,4404.51,19.34,320935.2280,4039005.1688,52,77.41,89.7353,0.155,0.287,0.000,-0.50
```
→ **Throttle 15% 감소** + **Steering 0.287 (우회전)**, 코너 감속 중

### 끝 부분 (안정 주행)
```csv
107.920,-10237.16,-2887.38,19.41,320730.2886,4039078.0877,52,77.13,-86.7192,0.159,-0.157,0.000,-0.31
```
→ 일정한 고속 유지, 좌측 조향

## 5. 시각화 (DrawDebug)

CSV 데이터는 동시에 월드에 시각화됨:

| 시각화 요소 | 트리거 |
|------------|--------|
| **트래젝토리 라인** | 매 100ms (Green ↔ Red HSV 보간) |
| **속도/Yaw 텍스트** | 5m마다 차량 위 표시 |
| **급감속 마커** | Magenta 점 (Acceleration < −50 km/h/s 시) |

> **이번 주행에서는 급감속 임계값(-50)에 도달한 케이스 0회**, 자율주행이 완만하게 코너 처리.

## 6. 데이터 품질 검증

### 샘플링 안정성
- 측정 평균 주기: **0.1001초** (예상 0.1초)
- 표준편차: ≤ 5ms
- → **10Hz 안정적 발행 확인**

### 좌표 변환 정확도
- WGS-84 UTM Zone 52 변환 적용
- 한국(36.48° N, 127.0° E) 기준 좌표
- UE 1cm = 1 UU → 0.01배 변환 후 UTM 적용

### 신뢰도
- ✅ 1,080행 모두 결측 없음
- ✅ 컬럼 13개 정상
- ✅ Acceleration 계산 식 검증 완료

## 7. 인사이트

### 자율주행 패턴 분석
1. **출발 직후 풀 스로틀** → 1.47 km/h에서 100% 가속 입력
2. **평균 62.75 km/h 유지** → 평균 32.5% 스로틀로 충분
3. **코너 감속은 엔진 브레이크 위주** → Brake 입력 0회
4. **조향 평균 0.332** → 코너 곡률에 따른 적절한 입력

이 데이터는 [`테스트결과.md`](../테스트결과.md)의 ch9 베스트 세팅보다 약 **2배 느린 랩타임**.

|------|-----------|----------|
| 랩타임 | 39.22초 | ~108초 (~2바퀴) |
| 평균 속도 | (높음) | 62.75 km/h |

**원인:** 새 BP는 ch9 튜닝값이 적용 안 된 디폴트 세팅 사용 (LookAheadBase 1500, MaxYawDelta 40 등)

## 8. 활용 방안

### 머신러닝 학습 데이터
- `(Yaw, Speed)` → `(Steering, Throttle)` 매핑 데이터셋
- 모방 학습(Imitation Learning) 입력으로 활용 가능

### Python 분석 예시
```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv('AgentData-Sample.csv')

# 트래젝토리 플롯
plt.scatter(df['UTM_Easting'], df['UTM_Northing'], 
            c=df['Velocity_kmh'], cmap='RdYlGn_r')
plt.colorbar(label='Speed (km/h)')
plt.axis('equal')
plt.show()
```
